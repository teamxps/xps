# Usage
<!-- usage -->
```sh-session
$ npm install -g @teamxps/xps
$ xps COMMAND
running command...
$ xps (-v|--version|version)
@teamxps/xps/0.0.0 win32-x64 node-v12.13.0
$ xps --help [COMMAND]
USAGE
  $ xps COMMAND
...
```
